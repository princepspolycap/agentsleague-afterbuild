# UI Assets

This folder is reserved for game art that is safe to ship with the public submission.

The local Polyverse office and interior packs can be used for private live demos, but their license files say the raw assets cannot be redistributed. Do not commit those PNGs here. Put private demo sprites under `submission/ui/assets/local/`; that path is ignored by Git.

When using the Modern Interiors pack in a local demo, credit `limezu.itch.io` as required by its license.
